# Arnav Jangle's Portfolio 🏑

A modern, next-gen portfolio website showcasing my field hockey journey and international tournament experience.

## 📋 Features

- ✨ Modern glassmorphism design with gradient accents
- 📱 Fully responsive (works on mobile, tablet, and desktop)
- ⚡ Smooth animations and transitions
- 🎯 Clean navigation with smooth scrolling
- 📊 About and achievements sections
- 💬 Contact section with links

## 📁 Project Structure

```
portfolio-arnav/
├── index.html          # Main HTML file
├── style.css           # Styling
├── script.js           # Interactivity
├── portfolio-photo.jpg # Your profile photo
└── README.md           # This file
```

## 🚀 How to Use

### 1. Add Your Photo
- Save your profile photo as `portfolio-photo.jpg` in this folder
- The photo will automatically appear in the hero section
- Recommended size: 400x500px or similar aspect ratio

### 2. Customize Your Info
Edit `index.html` to customize:
- Your email in the contact section (line 107)
- Your social media links (line 108)
- Any other personal details

### 3. Test Locally
Simply open `index.html` in your browser to preview the site.

## 🌐 Publish to GitHub Pages

### Step 1: Create a GitHub Repository
1. Go to [github.com](https://github.com) and sign in
2. Click the **+** icon and select "New repository"
3. Name it: `yourusername.github.io`
4. Make it Public
5. Click "Create repository"

### Step 2: Upload Your Files
1. Click "uploading an existing file"
2. Drag and drop your files:
   - `index.html`
   - `style.css`
   - `script.js`
   - `portfolio-photo.jpg`
   - `README.md`

### Step 3: Commit and Publish
1. Click "Commit changes"
2. Your site will be live at `https://yourusername.github.io`

## 💡 Tips

- Keep file names simple and lowercase
- Make sure your photo is in the same folder as the HTML files
- If the photo doesn't show, check the filename matches `portfolio-photo.jpg`
- You can update the content anytime by editing files in GitHub
